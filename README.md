# 📊 Communication Tracker App

A modern communication tracking web application built with **Next.js**. This app allows users to manage, track, and visualize their interactions such as calls, text messages, and video calls.

---

## 🚀 Features

### 🔹 Data Fetching

* Fetches data dynamically
* Clean and structured data handling

### 🔹 Error Handling

* Graceful error handling for failed API calls
* User-friendly error messages

### 🔹 Loading State

* Smooth loading UI while fetching data
* Improves user experience

### 🔹 Detailed View

* Click on any card to view detailed information
* Dynamic routing for individual items

### 🔹 Communication Actions

From the details page, users can:

* 📞 Make a Call
* 💬 Send a Text Message
* 🎥 Start a Video Call

---

## 🕒 Timeline Feature

* Displays all communication history
* Shows:

  * Who was contacted
  * Type of communication (Call, Text, Video)
  * Date of interaction

---

## 📈 Data Visualization

### Pie Chart Analytics

* Visual representation of communication types
* Shows:

  * Total number of Calls, Texts, and Video Calls

### Features:

* 🧾 Tooltip support
* 📌 Legend included for better understanding

---

## 🛠️ Tech Stack

* **Next.js**
* **React**
* **Context API (State Management)**
* **Tailwind CSS**
* **Chart Library (Pie Chart)**
